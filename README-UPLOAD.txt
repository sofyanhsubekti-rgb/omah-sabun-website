INSTRUKSI UPLOAD
1. Buka repo GitHub omah-sabun-website.
2. Upload semua file di folder ini ke root repository.
3. Replace file lama jika GitHub bertanya.
4. Commit changes.
5. Tunggu 1-3 menit, buka https://omahsabun.com, tekan Ctrl+F5.

Catatan:
- script.js tetap aman.
- CNAME/domain tidak diubah.
- Google Sheet katalog tetap aman.
