FAQ

Pertanyaan yang sering ditanyakan

Bisa order grosir?

Bisa. Kirim kebutuhan produk dan jumlah via WhatsApp — admin cek stok, harga, dan pengiriman untuk Anda.

Bisa jadi reseller?

Bisa. Isi form di atas agar admin menerima data awal dengan rapi, lalu akan di-follow up via WhatsApp.

Harga sudah final?

Harga di katalog sebagai referensi. Harga final tergantung jumlah, ukuran, dan area pengiriman — konfirmasi via WA admin.

Ada minimum order?

Tidak ada minimum untuk eceran. Untuk harga grosir dan reseller, tanyakan langsung ke admin via WhatsApp.
